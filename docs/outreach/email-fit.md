Madame, Monsieur,

J'ai appris la preparation de votre nouveau programme dedie a l'innovation sociale et je souhaite vous presenter **JusticePourtous.ch** comme projet candidat.

**En une phrase :** une plateforme open source de pre-conseil juridique gratuit qui rend le droit suisse accessible a ceux qui n'ont pas les moyens de consulter un avocat.

**Ou on en est (pas un slide deck, du code qui tourne) :**

- MVP fonctionnel et deploye en beta
- 40+ tests automatises, pipeline CI/CD operationnel
- Architecture : Node.js, LLM specialise sur le corpus juridique suisse, hebergement Suisse
- Open source, licence MIT, depot public
- Trois fonctions en production : diagnostic juridique, generation de courriers, orientation vers les structures existantes

**Le probleme en chiffres :**

- 72% des Suisses citent le cout comme barriere #1 a la justice
- CHF 250-450/h pour une consultation juridique
- 3 a 6 semaines de delai pour les permanences gratuites
- Resultat : les personnes precaires renoncent a faire valoir leurs droits

**Ce qu'on demande :**

Un financement de **CHF 50'000 a 80'000** sur 12 mois pour :
- Consolider la plateforme (securite, accessibilite, performance)
- Integrer le contenu juridique valide par des juristes professionnels
- Deployer un POC avec 2-3 organisations partenaires en Suisse romande (CSP Vaud en discussion)
- Mesurer l'impact reel (nombre de consultations, profil des utilisateurs, taux de resolution)

**Pourquoi la FIT :**

JusticePourtous est un projet technologique au service d'un impact social mesurable. L'innovation n'est pas dans le buzzword "IA" — elle est dans l'application concrete d'un LLM specialise a un probleme reel, avec un modele de distribution ouvert et un chemin vers l'autofinancement (abonnements pro pour les organisations, mandats d'integration cantonaux).

Le projet est base dans le canton de Vaud. L'equipe est locale. Le code est suisse.
