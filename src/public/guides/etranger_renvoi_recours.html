Délais critiques

  1. Analyse de la décision notifiée — délai : immédiat dès réception (LEI 64 / LEI 83)
  2. Consultation d'un mandataire qualifié — délai : sous 7 jours
  3. Dépôt du recours avec demande d'effet suspensif — délai : 30 jours (ou délai notifié) (LEI 112 / LAsi 108 / PA 55)
  4. Production des preuves d'intégration / risques — délai : pendant l'instruction (CEDH 3 / LEI 83 / LAsi 5)
  5. Décision de recours — délai : selon instance (LTAF 31 / CEDH 34)
